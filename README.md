# ATM Simulation (Python)

## Features
- PIN Authentication
- Balance Inquiry
- Deposit & Withdraw
- Transaction History
- Input Validation

## Technologies Used
- Python
- Functions
- Loops
- Dictionaries
- Lists

## Description
This project simulates basic ATM operations using Python. It is a menu-driven program that allows users to perform banking operations securely.
